# EE Explorer Extension
@RAWSEQ

for Windows (AuthHotKey)

# Usage

- Start ee.exe
- Push / Key on explorer.
- [Ctrl + h] Key to Help

# Keys
 [/]              - Next directory
 [Ctrl+Backspace] - Prev directory
 [Tab]            - Next suggestion
 [Shift + Tab]    - Prev suggestion
 [Ctrl + s]       - Set date
 [Ctrl + d]       - Create directory
